# MultiplyFun: Multiply By 2 Adventure! 🚀🌟

A fun, interactive, and bilingual (English/Afrikaans) web application designed to help Grade 2 students learn how to multiply by 2.

## Features
* **Learn to Multiply:** Visual circle groups ranging from `0 x 2` up to `12 x 2`.
* **Test Your Knowledge:** A randomized 13-question quiz with input fields and immediate visual/audio feedback, plus confetti for correct answers!
* **Bilingual Support:** Instantly toggle between English and Afrikaans.
* **Responsive Design:** Fully scales for desktop, tablet, and mobile viewing.

## Setup & Running Locally
Since this project uses plain HTML, CSS, and JavaScript with no build steps, you can run it instantly:

1. Clone or download the repository.
2. Open `index.html` in any modern web browser. 
3. Start learning!
